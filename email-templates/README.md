# Handlebars Email Templates Viewer

A simple web server to view and preview Handlebars email templates in your browser.

## Features

- List all available Handlebars templates
- Preview templates with sample data in the browser
- View raw template code

## Installation

1. Install dependencies:
   ```
   npm install
   ```

2. Start the server:
   ```
   npm start
   ```
   
   For development with automatic reloading:
   ```
   npm run dev
   ```

3. Open your browser and navigate to:
   ```
   http://localhost:3000
   ```

## Usage

- The home page displays a list of all available templates
- Click "Preview Template" to see the rendered template with sample data
- Click "View Raw HTML" to see the raw Handlebars template code
